About Your Weather

Your Weather (version 2) is a simple weather app that uses the OpenWeather API to get the current weather and 5 day forecast for any location in the world.

The user can enter any location they wish by entering the city or town name in the search box.

The user can toggle between metric and imperial units by clicking the temperature toggle switch, this will also change the wind speed from m/s to mph.

It was built using HTML, CSS and JavaScript and is fully responsive.

It was originally built by Boderg (me) for Milestone Project 2 of the Code Institute Full Stack Web Application Development course.

This version has been updated with the following over the original:

  • Weather Alerts
  • Revised layout
  • Re-structured HTML and CSS
  • Bootstrap Frontend
  • NodeJS Backend
